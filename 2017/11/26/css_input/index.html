额外的话

最外层的父元素div必须有宽度，而内层的input的宽度设置为100%。由于本文对所有元素使用了border-box效果，因此输入框的宽度最终取决于最外层父元素定义的宽度。当要实现多个表单元素的宽度一致的效果时，采用这种方式布局比较高效。

最外层的父元素div必须是相对定位relative，span元素则定位为absolute。如果定义了span的top|left|bottom|right却没有指定父元素的相对定位，span元素会以body为父元素去定位。

span元素和input的高度必须一致，这里使用的是line-height。line-height只对块级元素起作用。

span元素和input设置了字体大小，也需要一致，否则会出现字不在同一水平线上的情况。

适配各种浏览器

我们知道，不同浏览器厂商在生产浏览器的时候会有不同的默认样式。比如Safari，默认的margin为0，而其他浏览器的默认margin值是xxx，详情可查看浏览器默认样式对照表。为了尽可能地保证页面能够适配不同浏览器，我们需要重置样式表，也就是统一各个浏览器的默认样式，推荐的CSS Reset有 Normalise.css。
